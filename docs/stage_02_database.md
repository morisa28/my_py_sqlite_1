# 阶段 02：SQLite 数据库初始化

## 目标

实现需求文档中的数据库结构和初始数据，让系统首次运行时自动创建 SQLite 数据库。

## 完成内容

- 实现 `users`、`books`、`book_copies`、`borrow_records` 四张表。
- 实现数据库连接函数 `get_connection()`。
- 实现建表函数 `create_tables()`。
- 实现初始化函数 `init_db()`。
- 插入 1 个管理员账号。
- 插入 10 个普通用户账号。
- 插入 20 本初始图书。
- 为每本初始图书创建 3 条副本记录。
- 初始化逻辑支持重复执行，不会重复插入相同账号、图书或副本。

## 数据库文件

默认数据库文件为项目目录下的 `library.db`，由 `main.py` 或后续界面启动时自动生成。

## 验证重点

- 运行 `python3 main.py` 可以生成数据库。
- `users` 表至少包含 11 条记录。
- `books` 表至少包含 20 条记录。
- `book_copies` 表至少包含 60 条记录。
