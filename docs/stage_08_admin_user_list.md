# 阶段 08：管理员查看所有用户

## 目标

在管理员端增加用户总览能力，并支持从用户列表中直接查看指定用户的借阅记录。

## 完成内容

- 新增 `get_all_users()`：
  - 返回所有用户的用户 ID、账号、姓名、角色、状态、当前借阅数和历史借阅数。
  - 不查询、不返回密码字段。
- 新增 `get_user_borrow_records()`：
  - 根据用户 ID 精确查询该用户借阅记录。
  - 没有借阅记录时返回一条“无借阅记录”状态行，便于界面展示。
- 管理员界面新增“查看所有用户”按钮。
- 管理员在用户列表中右键某一行，可选择“查看借阅记录”。

## 用户表格字段

```text
用户ID
用户账号
用户姓名
角色
状态
当前借阅数
历史借阅数
```

## 验证重点

- 管理员可以查看所有用户。
- 用户列表不显示密码。
- 右键用户行可以切换到该用户借阅记录表。
- 没有借阅记录的用户会显示“无借阅记录”。
