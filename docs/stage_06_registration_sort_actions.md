# 阶段 06：注册、表格排序和右键操作

## 目标

改进图形界面的日常使用体验，加入用户注册、表格排序、查询后的恢复入口和行级右键操作。

## 完成内容

- 新增普通用户注册功能：
  - 登录页新增“注册”按钮。
  - 注册时填写账号、密码、确认密码和姓名。
  - 注册账号写入 `users` 表，角色固定为普通用户。
  - 账号重复、空字段、两次密码不一致时给出提示。
- 新增表格排序：
  - 图书表格默认按 `图书编号` 升序排列。
  - 点击任意列标题可按该列升序排序。
  - 重复点击同一列可在升序和降序之间切换。
- 新增“查看全部”按钮：
  - 普通用户和管理员左侧功能区均提供“查看全部”。
  - 点击后恢复显示所有未删除图书。
- 新增右键行操作：
  - 普通用户在图书表格中右键可借书。
  - 普通用户在“我的借阅”中右键可还书。
  - 管理员在图书表格中右键可修改图书信息。
  - 管理员在图书没有借出副本时可右键删除图书。

## 技术说明

- 新增 `ui/table_utils.py` 统一处理表格排序。
- `Treeview` 表头文字会显示当前排序方向。
- 右键菜单会先选中当前行，再根据当前表格类型和行数据生成可用操作。

## 验证重点

- 注册新用户后可立即登录。
- 重复账号注册失败。
- 密码确认不一致注册失败。
- 表格默认按图书编号升序排列。
- 点击表头可切换升序和降序。
- 查询图书后点击“查看全部”可恢复全量图书。
- 右键图书行可执行借书、修改或删除等当前可用操作。
