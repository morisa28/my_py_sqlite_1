# 阶段 05：README 与验证

## 目标

补齐项目交付说明，并为核心服务层增加可重复执行的自动化验证。

## 完成内容

- 完善 `README.md`：
  - 项目简介
  - 技术栈
  - 项目结构
  - 运行方法
  - 测试账号
  - 主要功能
  - 数据库说明
  - 注意事项
- 新增 `tests/test_services.py`。
- 测试使用临时 SQLite 数据库，不会污染正式 `library.db`。

## 测试覆盖

- 初始化数据数量。
- 管理员和普通用户登录。
- 错误密码登录失败。
- 用户借满 2 本后禁止继续借书。
- 用户还书后当前借阅数量更新。
- 管理员录入图书自动生成 3 个副本。
- 重复图书编号禁止录入。
- 修改图书时书名保持不变。
- 删除已借出图书被阻止。
- 管理员查询用户借阅状态。
- 图书总览和模糊查询。

## 验证命令

```bash
python -m unittest
```

## 环境说明

WSL 当前 Python 环境缺少 `tkinter` 模块，不能直接打开图形窗口。Windows Python 已验证可以导入 Tkinter，可使用 Windows Python 运行图形界面。
